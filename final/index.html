Проєкт на p5.js: Перевірка реакції

Огляд проєкту

Цей проєкт демонструє можливості p5.js. Він дозволяє користувачам взаємодіяти з графічними елементами в реальному часі.

Мета проєкту:

Створення інтерактивну гри для тренування/перевірки своєї реакції.

Основні завдання:

  • Певні UI елементи
  • Вибір режимів тестування
  • Забезпечення зворотного зв'язку з користувачем через події миші та клавіатури

Технічна реалізація

Використані технології:

  • p5.js для графіки та анімації
  • JavaScript для логіки взаємодії
  • HTML5/CSS3 для структури та стилізації

Інноваційні аспекти:

  • Інтерактивні елементи, що реагують на дії користувача
  • Взаємодіє гри з користувачем непередбачувано
  • Взаємодія, що змінюються в залежності від рандому

Особливості проєкту

  • Інтерактивні геометричні фігури
  • Генерація випадкових інтервалів часу між проявом об'єктів
  • Реакція на рух миші та клавіатуру
  • Зміна кольорів у реальному часі

Керування

  • Клік лкм - як дотик до об'єкта
